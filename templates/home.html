Projects

FlightHub

Python / HTML / CSS / Javascript / SQL / XAMPP

FlightHub is a website that helps users plan trips based on historical data of airfare prices. The goal of the website is to help travelers easily compare prices in the world of dynamic ticket pricing. The creation of FlightHub involved HTML, CSS, and Javascript in the frontend while Python, SQL, and XAMPP were used in the backend. In the future, gamification features and the ability to showcase trips will be added to differentiate the website.

TuneTalk

Swift / Storyboard

Tunetalk is a song recommender application built using Apple Music's API service. The project aims to augment algorithmic song recommendation with community elements such as reviews and ratings on how well songs are matched based on a variety of criteria (genre, artists, etc.). The project was fueled by my dissatisfaction with Spotify's lackluster 'Discover Weekly' feature for finding new music. Tunetalk encompasses the full stack development process, involving prototyping, design documentation, testing, and deployment.

Qommute

Python / Qiskit / PyTorch

Qommute is a Quantum Routing and Placement Project for a quantum computing hackathon! This project tackles the challenging task of optimizing bus station placements and routing using a combination of quantum computing and machine learning. By leveraging cutting-edge technologies, we can revolutionize public transportation systems for greater efficiency and reduced delays. As a resident of Queens, NY, improving the bus service is very important and needed. I am a victim of bus bunching and illogical service due to the MTA not having up-to-date information.

CreditCheck

Python / NumPy / Pandas / scikit-learn

CreditCheck is a project that explores how various machine learning techniques can or cannot accurately predict a person's credit risk based on their demographic data. Overall, the models were correct 93% of the time. Further exploration is needed to validate the results using additional datasets. From this project, I strengthened my Python skills and familiarized myself with two popular Python libraries for data analysis, NumPy and Pandas.
